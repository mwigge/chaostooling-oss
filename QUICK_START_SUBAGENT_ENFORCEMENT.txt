STEP 1: Create Unique Feature Branch
┌─────────────────────────────────────────────────────────────────┐
│ $ git checkout -b coder/feature/baseline-discovery              │
│ $ git push -u origin coder/feature/baseline-discovery            │
│                                                                  │
│ Branch naming: <agent>/<type>/<description>                     │
│ ✅ coder/feature/baseline-discovery                             │
│ ✅ tester/test/mongodb-tests                                    │
│ ✅ reviewer/security/sql-injection-audit                        │
│ ✅ indexer/docs/api-reference                                   │
│ ✅ orchestrator/ci/setup-actions                                │
└─────────────────────────────────────────────────────────────────┘

STEP 2: Develop Code Locally
┌─────────────────────────────────────────────────────────────────┐
│ # Write code with type hints                                    │
│ def discover_baseline(                                           │
│     system: str,                                                │

│     """Discover baseline metrics."""                            │
│     ...                                                         │

│     assert discover_baseline("postgres", {...}) == [...]        │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐

│                                                                 │
│ If fails → Fix locally and retry                               │
│ If passes → Commit                                             │
└─────────────────────────────────────────────────────────────────┘

STEP 4: Commit & Push
┌─────────────────────────────────────────────────────────────────┐

   Fix: Add tests to reach 95% coverage

   Fix: Get 2 code review approvals

  3. ✅ Don't push directly to main (protected)
  4. ✅ Wait for CI/CD to pass before review request
  5. ✅ Get 2 code approvals before merge
  6. ✅ Keep branch up to date with main

  1. ✅ Add type hints to all functions
